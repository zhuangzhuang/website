WiX

Summary

Contains functionality related to WiX.

In order to use the commands for this alias, include the following in your build.cake file to download and install from NuGet.org, or specify the ToolPath within the appropriate settings class: 

#tool "nuget:?package=WiX.Toolset"

Candle

WiXCandle(ICakeContext, IEnumerable<FilePath>, CandleSettings) Compiles all .wxs sources in the provided source files.
WiXCandle(ICakeContext, string, CandleSettings) Compiles all .wxs sources matching the specified pattern.

Heat

WiXHeat(ICakeContext, DirectoryPath, FilePath, WiXHarvestType) Harvests files in the provided object files.
WiXHeat(ICakeContext, DirectoryPath, FilePath, WiXHarvestType, HeatSettings) Harvests files in the provided directory path.
WiXHeat(ICakeContext, IEnumerable<FilePath>, FilePath, WiXHarvestType) Harvests from the desired files.
WiXHeat(ICakeContext, IEnumerable<FilePath>, FilePath, WiXHarvestType, HeatSettings) Harvests from the desired files.
WiXHeat(ICakeContext, string, FilePath, WiXHarvestType) Harvests files for a website or performance.
WiXHeat(ICakeContext, string, FilePath, WiXHarvestType, HeatSettings) Harvests files for a website or performance.

Light

WiXLight(ICakeContext, IEnumerable<FilePath>, LightSettings) Links all .wixobj files in the provided object files.
WiXLight(ICakeContext, string, LightSettings) Links all .wixobj files matching the specified pattern.
